# WORKORDER-02 — UI 시스템 전면 통일 (웜 그레이지 · 거의 무채색)

> 대상: Windows Claude Code. **프론트엔드 전용**(백엔드/DB/인증/RLS/스키마 변경 없음 → CLAUDE.md상 Codex 필수 아님. 단 `next build` + 화면 스크린샷 검증 필수).
> 근거: PM방 UI 확정(2026-07-23). 랜딩 감사(`docs/landing_diagnosis_audit.md`)와 함께 참고.
> 원칙 한 줄: **색은 거의 없다. 흰 배경 + 연한 웜그레이 + 소프트 차콜. 화려함은 사진이 담당한다.**

## 🎯 목표

전 화면(랜딩 4종 + 홈 + 커머스 + 업로드/결과 + 정책 페이지)을 **하나의 디자인 토큰 세트**로 통일한다. 현재의 "화면마다 다른 규칙(크림/골드 남발, 순검정 업로드, 청록 버튼, 무지개+이모지 하단탭)"을 제거하고 브랜드 일관성을 만든다.

## 🎨 디자인 토큰 (SSOT — tailwind.config 또는 전역 CSS 변수로 1곳에 정의 후 전 컴포넌트가 참조)

```
/* 색 — 웜 그레이지, 무채색 */
--bg:        #FFFFFF;   /* 페이지 배경 (크림 폐지) */
--surface:   #F5F2ED;   /* 연한 웜그레이 면 (pill, 구분 섹션) */
--card:      #FFFFFF;   /* 카드 (테두리로 구분) */
--line:      #EAE5DD;   /* 경계선 */
--ink:       #33302C;   /* 본문 텍스트 (진검정 아님) */
--ink-2:     #8B8378;   /* 보조 텍스트 (이보다 흐리게 금지) */
--ink-3:     #B7B0A5;   /* 힌트 (아주 드물게) */
--btn:       #3B3733;   /* 소프트 차콜 = 주 CTA 채움 */
--btn-line:  #DAD5CD;   /* 아웃라인 버튼 테두리 */
/* 유채색 액센트 없음. 진행바/활성탭은 --ink 사용 */

/* 형태 */
--r-card:16px; --r-btn:12px; --r-pill:999px;
--shadow: 0 3px 12px rgba(30,27,24,.05);   /* 1단계만, 남발 금지 */
--pad-x:20px;   /* 화면 좌우 패딩 고정 */
/* 간격: 8pt 그리드(8·16·24·32) */
```

## 🔠 타이포 (Pretendard)
- H1 26px/800 · H2 18px/800 · 본문 16px/400 `#33302C` · 보조 13px/500 `#8B8378`
- **위계로 승부**: 화면당 강조(볼드/큰 크기)는 1곳만. 지금처럼 제목·부제·버튼이 다 크고 굵은 것 금지.

## 🔘 버튼 (3종)
- **Primary(화면당 1개 주 CTA만)**: `--btn`(#3B3733) 채움 + 흰 텍스트. 예: 랜딩 "분석하기", "1분 자가진단 시작하기".
- **Secondary**: 투명 + `--btn-line` 테두리 + `--ink` 텍스트. (기존 밑줄 텍스트버튼 폐지)
- **Ghost**: 텍스트만.
- Disabled는 `--surface` 배경 + `--ink-3` 텍스트로 **명확히 구분**.

## 🧭 아이콘
- **Lucide 한 세트로 통일**, stroke 1.8, 단색(`--ink-2`, 활성 `--ink`). **UI에서 이모지 전면 제거**(🔥📋✍️ 등).
- 하단탭: 무지개색 폐지 → 단색 라인, **활성 탭만 `--ink`로 진하게**(색으로 구분 금지).

## 🖼 이미지 우선
- 카드·결과지에 **이미지 슬롯을 레이아웃에 반영**(지금 비어 보이는 이유 = 이미지 부재). `/style` Before/After 합성 사진을 히어로로 크게. 화려함은 색이 아니라 사진이 만든다.

## 📄 화면별 필수 교정
- **`/style/upload` 순검정 → 웜 차콜**(`#1A1714` 그라데이션). 점선 얼굴 가이드 → 부드러운 실루엣. **"합성 후 즉시삭제·원본 미보관" 안심문구를 크게·고대비**로 앞에(50·60 이탈 방지). 촬영 가이드 이미지 자산은 유지.
- **`/home`**: 청록 "내 고민 등록하기" 버튼 → Primary(소프트 차콜) 또는 Secondary. 고민상담소 카드·pill을 토큰 규칙으로. 이모지 아이콘 제거.
- **랜딩 4종(`/style`·`/bangs`·`/damage-check`·`/hair-quiz`)**: 크림 배경 → 흰 배경, 골드 남발 제거, 헤드라인 크기 축소, CTA는 Primary 1개 규칙.
- **`/items`·`/items/[id]`·`/my-diary`·`/privacy`·`/terms`**: 동일 토큰 적용(색 통일, pill/카드/버튼 규칙 재사용).
